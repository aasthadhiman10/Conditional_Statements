# Conditional-Statement

This repository is dedicated to mastering **Conditional Logic** and **Control Flow** in Java. Instead of simple calculations, this project focuses on how a program makes decisions based on user input and various conditions.

## 🚀 Projects Included

* **If_Else_Statement.java**: A foundational program to understand basic conditional branching using age verification logic.
* **If_Else_Statement_2.java**: A logical program that checks whether a given integer is an even or odd number using mathematical conditions.
* **Else_If_Statement.java**: Implements an "Else-If Ladder" to compare two dynamic numbers (`a` and `b`) and find which one is greater or if they are equal.
* **Else_If_Statement_2.java**: Uses a multi-conditional setup to route user choices (buttons) to specific language configurations.
* **Switch_Break_Statement.java**: A robust, menu-driven setup that manages multiple button options efficiently using `switch-case` and includes error handling via a `default` case.

## 🛠️ Concepts Covered

* **Decision Making**: `if`, `else if`, `else` constructs to handle multiple execution branches.
* **Branching & Optimization**: `switch-case` statements with `break` implementation for handling exact matches cleanly.
* **Input Handling**: Using `java.util.Scanner` for interactive, dynamic user experiences at runtime.
* **Defensive Programming**: Resource management via `sc.close()` to prevent memory leaks and handling wrong inputs using the `default` switch block.
